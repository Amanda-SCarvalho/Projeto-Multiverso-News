# 🌐 MyTech Hub

O **MyTech Hub** é um portal web desenvolvido como parte das atividades do **PROA**, com o objetivo de aplicar na prática conceitos de desenvolvimento web utilizando **HTML e CSS**.

O projeto simula um portal de notícias inspirado no universo geek, reunindo conteúdos sobre tecnologia, cultura nerd, filmes, jogos e inovação.

## 🌍 Acesse o Projeto

Você pode acessar o site publicado aqui:

➡️ [Link do site](https://amanda-scarvalho.github.io/Projeto-Multiverso-News/)

## 🚀 Objetivo do Projeto

Criar um portal de notícias simples e organizado, aplicando conceitos fundamentais de desenvolvimento web, como:

* Estruturação semântica com **HTML**
* Estilização e layout com **CSS**
* Organização de arquivos e pastas
* Boas práticas de **SEO**
* Princípios de **acessibilidade**

## 📰 Multiverso News

Dentro do projeto foi criado o **Multiverso News**, um portal fictício de notícias geek onde os usuários podem explorar conteúdos sobre diferentes universos da cultura pop.

O site apresenta:

* Página inicial com destaques
* Cards de notícias
* Seção de reviews de tecnologias
* Layout inspirado no universo geek

## 🛠 Tecnologias Utilizadas

* HTML5
* CSS3
* Google Fonts

## 🌍 Publicação

O projeto pode ser publicado utilizando **GitHub Pages**, permitindo que o site fique disponível online.
