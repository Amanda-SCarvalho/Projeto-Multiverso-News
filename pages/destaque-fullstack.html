ClipMaker

Stack: HTML, Tailwind CSS, JavaScript, Gemini API, Cloudinary

Educador: Mayk Brito

Sobre o Projeto

O ClipMaker é uma aplicação web que utiliza Inteligência Artificial para gerar automaticamente cortes virais a partir de vídeos longos. A plataforma analisa transcrições do conteúdo utilizando a Gemini API e identifica os momentos mais impactantes do vídeo.

Após a análise, o sistema seleciona o trecho ideal e exibe o clipe gerado em tempo real através da integração com o Cloudinary, permitindo uma visualização rápida e prática do conteúdo destacado.

Este projeto faz parte da trilha iniciante para quem deseja aprender programação com IA, explorando conceitos fundamentais de desenvolvimento web enquanto constrói uma aplicação moderna e funcional.

Funcionalidades

  • ✔ Upload e análise de vídeos longos
  • ✔ Identificação automática de momentos virais com IA
  • ✔ Integração com Gemini para análise de transcrição
  • ✔ Processamento e entrega de clipes com Cloudinary
  • ✔ Interface moderna utilizando Tailwind CSS
  • ✔ Visualização do clipe gerado em tempo real
